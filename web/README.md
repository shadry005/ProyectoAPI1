# Web Root

## Estado actual
Este directorio contiene una estructura inicial para un frontend, pero los archivos aún no están implementados.

## Estructura
- `core/`
  - `urls.py`
  - `views.py`
- `src/`
  - `app.ts`
  - `controllers/`
  - `models/`
  - `views/`
- `package.json` (actualmente vacío)

## Cómo continuar
1. Define las dependencias necesarias en `package.json`.
2. Implementa la lógica de frontend en `src/app.ts` y los controladores.
3. Usa `core/urls.py` y `core/views.py` si quieres un backend Django o un punto de entrada para la aplicación.

## Sugerencias
- Si el proyecto es Django, añade la configuración del proyecto y las apps necesarias.
- Si es una app Node/TS, añade script de compilación y arranque.

## Notas
Este README documenta el estado actual del directorio `web/` y provee una guía básica para continuar su desarrollo.
